# Sales & Customer Analytics Dashboard (Databricks + Power BI)  
## Overview  

This project demonstrates end-to-end data exploration and dashboard creation using Databricks and Power BI.

The workflow covers:
- Exploring structured data using SQL 
- Building visualizations in Databricks  
- Creating business dashboards in both Databricks and Power BI  
  
The goal is to simulate a real-world analytics pipeline from raw data → insights → stakeholder-ready dashboards.

### Data Exploration (SQL Editor)  
- Understand the dataset and become familiar with the Databricks SQL environment.  

### Visualization (Databricks Dashboards)
- Convert SQL queries into meaningful visual insights.  

### Business Dashboards (Databricks)
- Sales Dashboard
- Customer Dashboard
  
  ### Features
  - KPI cards
  - Time-series analysis
  - Category breakdowns
  - Global + local filters

**Power BI Dashboard**
- Recreate and enhance dashboards using Power BI for richer interactivity and presentation.

## Key Skills Demonstrated

- SQL analytics (aggregations, joins, window functions)
- Data exploration
- Dashboard design (Databricks + Power BI)
- Business metric development
- Data storytelling
- Cross-platform BI integration

## Future Improvements

- Add cohort analysis (customer retention)
- Implement advanced KPIs (LTV, churn rate)
- Optimize SQL queries for large-scale performance
- Deploy dashboards for stakeholder access
